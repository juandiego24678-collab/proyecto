Seguridad

Propuesta de Seguridad

Medidas integrales para proteger la red, los sistemas y los datos del coworking

Seguridad en la Red

  • 🛡️ Router/Firewall Ubiquiti UniFi para filtrar y controlar el tráfico de red
  • 🔀 Switch gestionable TP-Link para segmentar y controlar el tráfico entre dispositivos
  • 🔒 Wi-Fi segura con cifrado WPA3
  • 👤 Control de accesos mediante usuarios autorizados
  • 🔄 Actualización periódica del firmware de los dispositivos de red

Seguridad en los Sistemas

  • 🦠 Bitdefender GravityZone (antivirus empresarial en todos los equipos)
  • 🏢 Active Directory (AD DS) para gestión centralizada de usuarios y GPOs
  • 🚫 Eliminación de servicios innecesarios para reducir vulnerabilidades
  • 💾 Copias de seguridad automáticas con Veeam Backup & Replication en NAS
  • 🔁 Actualización constante del SO y software instalado

Documentación Técnica

Justificación Técnica

Razón técnica de cada elección de hardware, red y software del proyecto

1. Hardware y Equipos de Usuario

Equipos Dell OptiPlex 3060 SFF

Seleccionados por su fiabilidad en entornos profesionales, bajo consumo energético y formato compacto, optimizando el espacio en las mesas. Con i5, 16GB RAM y SSD 512GB ofrecen rendimiento fluido para ofimática, videoconferencias y multitarea.

Periféricos y Reprografía

Monitor, teclado y ratón independientes en cada puesto para una configuración flexible y ergonómica. La HP LaserJet Pro MFP cubre impresión, escaneo y copiado para todo el espacio como recurso de red compartido.

2. Infraestructura de Red y Servidores

Servidor HP ProLiant + Windows Server 2022

Permite crear un dominio mediante Active Directory (AD DS), gestionando los 14 equipos de forma centralizada: credenciales, GPOs, DHCP y unidades de red automáticas desde un único punto de control.

NAS Synology DS923+

Proporciona almacenamiento compartido y copias de seguridad automatizadas con Veeam, mejorando la disponibilidad de la información y reduciendo el riesgo de pérdida de datos.

Switch TP-Link + Router Ubiquiti + SAI

El switch gestionable garantiza conexión estable entre dispositivos. El router/firewall protege frente a accesos no autorizados. El SAI APC 1500VA asegura la continuidad del servicio ante cortes eléctricos.

3. Software y Seguridad

Windows 11 Pro

Elegido por su compatibilidad con herramientas empresariales y funciones avanzadas de seguridad (TPM 2.0, Secure Boot, Bitlocker). Soporte nativo para RDP y RSAT para administración remota.

Microsoft 365 Business Standard

Incluye Teams para comunicación y colaboración, OneDrive para almacenamiento en la nube y toda la suite de productividad. 14 licencias para todos los puestos del coworking.

Bitdefender GravityZone + Veeam

Antivirus empresarial con gestión centralizada en todos los equipos. Veeam Backup realiza copias automáticas al NAS, reduciendo significativamente el riesgo de ciberataques y pérdidas de información.

Herramienta y proceso

El sistema se gestiona mediante GLPI instalado en el servidor central, o en su defecto, un documento compartido en el NAS Synology.

Cuando un coworker detecta un fallo (conectividad, error de impresión, etc.), genera un ticket con su prioridad. El técnico actualiza el estado a "En proceso" y documenta los pasos técnicos al cerrar el ticket.

Este sistema crea una base de conocimientos que permite resolver problemas similares mucho más rápido en el futuro.
