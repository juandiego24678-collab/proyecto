03

Configuración de Servicios de Red

La red fue diseñada para proporcionar conectividad estable y segura a todos los dispositivos, utilizando el rango privado 192.168.1.0/24.

3.1 Dirección IP

Modo: DHCP automático para equipos cliente
Rango general: 192.168.1.0/24
Gateway: 192.168.1.1 (Router/Firewall Ubiquiti)
Servidor Central: 192.168.1.10
Seguridad Wi-Fi: Cifrado WPA3

3.2 Router y Switch

Router/Firewall: Ubiquiti UniFi Security Gateway (DHCP, NAT, Firewall)
Switch: TP-Link TL-SG1024DE (Gigabit, puertos no usados deshabilitados)

3.3 DNS y Asignaciones Fijas

Servidor Central (Windows Server 2022): 192.168.1.10
Sistema NAS (Synology): 192.168.1.20
Impresora (HP LaserJet): 192.168.1.30

Pruebas Técnicas Realizadas

Una vez configurada la infraestructura, se realizaron pruebas de conectividad y funcionamiento utilizando Cisco Packet Tracer.

5.1 Pruebas de conectividad y servicios

  • Comunicación entre todos los dispositivos mediante ping
  • Acceso al servidor y a las carpetas compartidas validado
  • Funcionamiento de la impresora de red verificado
  • Acceso a Internet desde los equipos cliente funcional
  • Verificación del correcto funcionamiento del DHCP y DNS

5.2 Conclusión del despliegue

✓
No se detectaron incidencias

La infraestructura configurada proporciona un entorno de coworking estable, seguro y preparado para un uso profesional. Todas las pruebas se realizaron correctamente.
